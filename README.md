## 🚀 Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/yourusername/nabin-portfolio

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
streamlit run app.py
```

Then open `http://localhost:8501` in your browser.
